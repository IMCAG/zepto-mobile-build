# zepto-mobile-build
Custom [Zepto](https://github.com/madrobby/zepto) build replacing jQuery in some IMC mobile apps.

Included modules: 
- zepto
- event
- ie
- data
- selector
- touch
- gesture
- stack

## Comparison

:-- | -----------: | ---------:
jQuery 3 | 271 KB | 85 KB
jQuery 2 | 242 KB | 83 KB
Zepto | 56 KB | 26 KB
